# Properly Hired — Simple User Guide (Ages 50+)

This guide is designed to be easy and calm. Follow the steps in order. You do not need to explore every page.

## The 3 Things This System Does
1. Analyze a job posting.
2. Improve your resume for that job.
3. Write a cover letter.

Everything else is optional or advanced.

## Quick Start (Most People Use Only These 3 Pages)
1. **Job Analysis**
   - Paste the job link or paste the job description.
   - Click **Analyze**.
   - You will get key requirements and a recommendation score.
   - Simon (Recruiting Expert) is available on this page.

2. **Resume Optimizer**
   - Select your resume and the job you analyzed.
   - Click **Optimize**.
   - Review the improved version.
   - Kyle (Resume Expert) is available on this page.

3. **Cover Letters**
   - Select the job you analyzed.
   - Generate a cover letter.

If you only do these three steps, you are using the system correctly.

## Recommended Step‑By‑Step Flow (Logical Order)
1. **Create or Upload Resume**
   - Go to `Resume Builder` or `Resume Editor`.
   - Upload or enter your resume.

2. **Analyze a Job**
   - Go to `Job Analysis`.
   - Paste the job URL or description.
   - Save the analysis if prompted.

3. **Optimize Resume**
   - Go to `Resume Optimizer`.
   - Select the job you just analyzed.
   - Generate improvements.

4. **Create a Cover Letter**
   - Go to `Cover Letters`.
   - Select the job.
   - Generate the letter.

5. **Track Your Applications (Optional)**
   - Go to `Application Tracker`.
   - Save and track application status.

## Page‑By‑Page Map (What Each Page Is For)
Below is a simple description for every page. If it is marked **Advanced**, most users can skip it.

### Core Pages (Use These)
- `Dashboard`: Your starting screen.
- `Job Analysis`: Analyze a job posting. Simon appears here.
- `Resume Optimizer`: Improve your resume. Kyle appears here.
- `Cover Letters`: Generate cover letters.
- `Resume Builder`: Create a resume from scratch.
- `Resume Editor`: Edit your resume.
- `Resume Templates`: Choose a template style.
- `Resume Viewer`: View final resume.
- `Job Library`: Saved jobs list.
- `Job Details`: Details for a saved job.
- `Job Summary`: Summary after analysis.
- `Application Tracker`: Track applications.

- `Networking Hub`: Networking advice.
- `My Network`: Saved contacts.
- `Recruiter Connect`: Recruiter guidance.
- `People Search`: Search for people (if enabled).

### Advanced / Admin (Usually Hidden)

## Who the Agents Are (Simple)
- **Kyle** = Resume expert.
- **Simon** = Job/recruiting expert.
- **CV Assistant** = General helper that routes to Kyle or Simon.

## Common Questions
**Q: Do I have to use all pages?**
No. Most users use only Job Analysis, Resume Optimizer, and Cover Letters.

**Q: What if I don’t understand a result?**
Use the chat button on that page. Kyle or Simon will explain it.

**Q: Where do I start?**
Start at `Job Analysis` or `Resume Builder` if you need a resume.

## KISS Checklist (Print This)
1. I have a resume.
2. I analyzed the job.
3. I optimized the resume.
4. I created a cover letter.
5. I tracked the application.

If you completed these, you’re done.

## Notes for Admins (Keep Simple for Users)
- Keep the sidebar focused on the 3 core tasks.
- Move advanced tools under **Admin** only.
- Do not show the guided tour until a user opens Job Analysis, Resume Optimizer, or Cover Letters.
